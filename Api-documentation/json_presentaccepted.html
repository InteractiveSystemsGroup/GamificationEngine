PresentAccepted Data Type

Presents that are in the in-box of the board a player can accept or deny. If the player accept a present its status is set to accepted and an PresentAccepted object is created.

Properties
name data type description
date LocalDateTime Gets the date and time a present was accepted by a player.
id number Gets the id of the accepted present.
present Present Get the present that is accepted by the player.
status Status Sets the status of the present if the player has accepted it or not.
